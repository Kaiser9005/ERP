# Support de Formation - Analytics RH

## Introduction

Ce support de formation est destiné aux utilisateurs finaux du module Analytics RH. Il propose une approche pratique pour maîtriser l'utilisation quotidienne des tableaux de bord et des analyses RH.

## Objectifs de la Formation

À la fin de cette formation, vous serez capable de :
1. Naviguer efficacement dans le tableau de bord Analytics RH
2. Interpréter les indicateurs clés
3. Utiliser les fonctionnalités avancées
4. Prendre des décisions basées sur les données

## Programme Détaillé

### Module 1 : Prise en Main (1h)

#### 1.1 Connexion et Navigation
- Accès au module Analytics RH
- Présentation de l'interface
- Navigation entre les différentes sections

#### 1.2 Exercice Pratique
1. Connectez-vous à l'application
2. Accédez au module Analytics RH
3. Identifiez les différentes sections du tableau de bord
4. Naviguez entre les différentes vues

### Module 2 : Les Indicateurs Clés (2h)

#### 2.1 Statistiques des Employés
- Effectif total et variations
- Présences et absences
- Formations en cours
- Tendances et évolutions

#### 2.2 Exercice Pratique - Analyse des Effectifs
1. Analysez l'évolution de l'effectif sur le dernier mois
2. Identifiez les tendances des présences
3. Évaluez le taux de participation aux formations
4. Rédigez une synthèse des observations

#### 2.3 Cas Pratique - Scénario 1
"L'effectif présent aujourd'hui est en baisse de 15%. Analysez la situation et proposez des actions."

1. Consultez les indicateurs pertinents
2. Identifiez les causes possibles
3. Proposez des mesures correctives
4. Documentez votre analyse

### Module 3 : Analyse des Formations (2h)

#### 3.1 Indicateurs de Formation
- Taux de complétion
- Répartition par type
- Taux de réussite
- Tendances

#### 3.2 Exercice Pratique - Analyse des Formations
1. Analysez le taux de complétion global
2. Identifiez les formations les plus suivies
3. Évaluez les taux de réussite
4. Proposez des améliorations

#### 3.3 Cas Pratique - Scénario 2
"Le taux de réussite d'une formation importante est en baisse. Analysez et proposez des solutions."

1. Examinez les données historiques
2. Identifiez les facteurs possibles
3. Proposez un plan d'action
4. Présentez vos conclusions

### Module 4 : Gestion des Contrats (1h30)

#### 4.1 Analyse des Contrats
- Types de contrats
- Durées moyennes
- Taux de renouvellement
- Alertes et notifications

#### 4.2 Exercice Pratique - Gestion des Contrats
1. Analysez la répartition des types de contrats
2. Évaluez les taux de renouvellement
3. Identifiez les contrats à surveiller
4. Créez un rapport de synthèse

### Module 5 : Analyse de la Masse Salariale (1h30)

#### 5.1 Indicateurs Financiers
- Masse salariale totale
- Salaire moyen
- Distribution des salaires
- Tendances

#### 5.2 Exercice Pratique - Analyse Financière
1. Analysez l'évolution de la masse salariale
2. Comparez avec les objectifs
3. Identifiez les écarts significatifs
4. Proposez des ajustements

## Exercices d'Application

3. Proposez des solutions
4. Présentez vos résultats
